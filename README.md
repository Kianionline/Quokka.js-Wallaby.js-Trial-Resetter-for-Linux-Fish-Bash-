# Quokka.js-Wallaby.js-Trial-Resetter-for-Linux-Fish-Bash-
This script resets the 15-day trial period for Quokka.js and Wallaby.js in JetBrains IDEs (WebStorm) by removing local timestamp files.
---

```markdown
# 🚀 راهنمای جامع و صفر تا صد استفاده نامحدود از Quokka.js در لینوکس

این راهنما شامل مراحل نصب، راه‌اندازی اولیه و ترفند صفر کردن همیشگی تایمر افزونه Quokka.js در وب‌استورم (WebStorm) است.

---

## 1️⃣ مرحله اول: نصب Quokka.js در وب‌استورم
اگر افزونه را هنوز نصب نکرده‌اید یا آن را پاک کرده‌اید، مراحل زیر را بروید:
1. وب‌استورم را باز کنید.
2. از منوی بالا به مسیر `File` > `Settings` (در مک: `WebStorm` > `Preferences`) بروید.
3. در منوی سمت چپ، روی **Plugins** کلیک کنید.
4. در تب **Marketplace**، عبارت `Quokka` را سرچ کنید.
5. روی دکمه **Install** جلوی افزونه **Quokka.js** کلیک کنید و منتظر بمانید تا نصب شود.
6. اگر وب‌استورم خواست که IDE را ری‌استارت کند، آن را ری‌استارت کنید.

---

## 2️⃣ مرحله دوم: چطور از Quokka استفاده کنیم؟
برای اینکه کدهایتان زنده اجرا شوند، باید افزونه را روی فایل جاوااسکریپت خود استارت کنید:
1. یک فایل جاوااسکریپت جدید بسازید (مثلاً `test.js`).
2. چند خط کد ساده بنویسید؛ مثلاً:
   ```javascript
   const a = 10;
   const b = 20;
   console.log(a + b);

```

3. دکمه‌های ترکیبی **`Ctrl + Shift + P`** (یا در بعضی نسخه‌ها `Ctrl + Shift + A`) را بزنید تا منوی دستورات وب‌استورم باز شود.
4. عبارت **`Quokka.js: Start on Current File`** را تایپ کنید و اینتر بزنید.
5. **نتیجه:** حالا جلو یا زیرِ خط `console.log`، خروجی کد را به صورت زنده (عدد ۳۰) خواهید دید!

---

## 3️⃣ مرحله سوم: دور زدن محدودیت ۱۵ روزه (Trial Reset)

بعد از ۱۵ روز، افزونه قفل می‌شود و از شما لایسنس می‌خواهد. برای اینکه تایمر آن را صفر کنید و دوباره ۱۵ روز رایگان بگیرید، مراحل زیر را بروید:

1. **ابتدا نرم‌افزار وب‌استورم را کاملاً ببندید.**
2. ترمینال لینوکس خود (Fish Shell) را باز کنید.
3. این ۳ دستور را به ترتیب وارد کنید و اینتر بزنید:

```bash
# ۱. رفتن به محیط بش برای جلوگیری از ارورهای ستاره (*) در پوسته فیش
bash

# ۲. حذف بی‌رحمانه فایل‌های مخفی تایمر و کش کووکا
rm -rf ~/.quokka ~/.wallaby ~/.config/JetBrains/WebStorm*/options/quokka.xml

# ۳. خروج و بازگشت به محیط اصلی ترمینال شما
exit

```

4. حالا دوباره وب‌استورم را باز کنید و طبق **مرحله دوم** افزونه را استارت کنید. کووکا فکر می‌کند افزونه تازه برای اولین بار نصب شده است و ۱۵ روز دیگر به شما وقت می‌دهد!
